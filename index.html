Привет друг,проверим твою внимательность?

Твоя задача найти две одинаковые карты, совершив наименьшее количество попыток.

Правила:

  1. Кликни по двум картам,чтобы найти одинаковые.
  2. Если изображения совпадают, ты молодец! Продолжай дальше.
  3. Если изображения не совпадают, повтори попытку.
  4. Игрок который найдет все изображения побеждает.
  5. Удачи!
